# Laravel API Shop

Este es un proyecto API construido con Laravel, utilizando PostgreSQL como base de datos. El proyecto incluye autenticación, una tabla de productos y un sistema de carrito de compras.

## Requisitos

- PHP >= 8.0
- Composer
- PostgreSQL
- Docker (opcional)
